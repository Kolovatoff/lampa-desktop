## 🚀 Быстрый старт

### 1. Установка парсера

После запуска приложения откройте настройки Lampa:
Настройки → Парсер → Указать: jac.red

### 2. Настройка плеера

Для просмотра видео необходим внешний плеер.  
Перейдите в раздел **Настройки → Плеер** и выберите установленный плеер.

> **Плееры с поддержкой тайм-кодов:**
>
> - VLC (для всех ОС)
> - KMPlayer / MPC-HC / MPC-BE / MPC-QT (Windows)
> - MPC-QT (Linux)
> - IINA (macOS)

### 3. Запуск TorrServer

Перейдите в: Настройки → Приложение → TorrServer

Нажмите **▶️ Запуск TorrServer**.  
Приложение автоматически скачает нужную версию с [GitHub](https://github.com/YouROK/TorrServer) и запустит её.

### ✅ Готово!

Теперь можно пользоваться приложением для просмотра контента.

## 🎯 Установка и настройка плеера

### Автоматический поиск плеера

1. Откройте `Настройки → Плеер → Поиск и выбор плеера`
2. Выберите найденный плеер из списка
3. Настройка завершена

### Ручная настройка (если плеер не найден)

1. Откройте `Настройки → Плеер → Путь к плееру`
2. Вручную укажите путь к исполняемому файлу плеера

### Для пользователей macOS

1. Установите [IINA](https://iina.io/)
2. Откройте `Настройки → Плеер → Тип плеера для торрентов`
3. Выберите **IINA**

## ⚙️ Настройка TorrServer

TorrServer необходим для просмотра торрент-контента.

### Запуск

Настройки → Приложение → TorrServer → ▶️ Запуск TorrServer

### Управление

| Кнопка                        | Действие                              |
| ----------------------------- | ------------------------------------- |
| ▶️ Запуск                     | Скачать и запустить TorrServer        |
| 🛑 Остановка                  | Остановить работу TorrServer          |
| 🔁 Перезапуск                 | Перезапустить TorrServer              |
| 🔍 Проверка обновлений        | Проверить наличие новой версии        |
| 📂 Открыть папку              | Открыть папку с файлами TorrServer    |
| 🌐 Открыть веб                | Открыть веб-интерфейс TorrServer      |
| 🗑️ Удалить (полностью)        | Полное удаление TorrServer            |
| 💾 Удалить (сохранить данные) | Удалить программу, сохранив настройки |

### Автозапуск

Включите **Автозапуск при старте Lampa** — TorrServer будет запускаться автоматически вместе с приложением.

## 🔧 Устранение проблем

### 1️⃣ Не работают балансировщики / плагины для онлайн просмотра

**Решение:** Отключите проверку CORS
Настройки → Приложение → Проверка CORS → Нет

> **Что такое CORS?** Это механизм безопасности браузера. При отключении вы действуете на свой риск, но это часто помогает при проблемах с балансировщиками.

---

### 2️⃣ Плеер не найден автоматически

**Возможные причины:**

- Плеер не установлен
- Плеер установлен в нестандартную папку

**Решение:**

1. Установите один из поддерживаемых плееров
2. Либо укажите путь к плееру вручную в настройках

---

### 3️⃣ TorrServer не запускается

**Проверьте:**

- ✅ Не занят ли порт 8090 (или порт, указанный в настройках)
- ✅ Есть ли доступ к интернету для скачивания
- ✅ Наличие прав на запись в папку приложения

---

### 4️⃣ Видео не воспроизводится

**Решение:**

1. Убедитесь, что плеер правильно выбран в настройках
2. Проверьте, запущен ли TorrServer (должен быть статус "✅ Запущен")
3. Перезапустите TorrServer кнопкой "🔁 Перезапуск"

---

### 5️⃣ Нет звука

**Решение:**
Убедитесь, что вы используете внешний плеер. Т.к. во внутреннем плеере нет всех необходимых кодеков.

---

### 6️⃣ Окно не сохраняет позицию (Linux Wayland)

На некоторых дистрибутивах Linux с Wayland позиция окна может не сохраняться.  
Это ограничение протокола Wayland. Рекомендуется использовать X11 для полной функциональности.

## ❓ Часто задаваемые вопросы

### Вопрос: Нужно ли каждый раз запускать TorrServer вручную?

**Ответ:** Нет, можно включить **Автозапуск при старте Lampa** в настройках TorrServer.

### Вопрос: Как перенести настройки на другое устройство?

**Ответ:** Используйте **Экспорт/Импорт настроек** в разделе `Приложение → Экспорт/Импорт настроек`. А также инструкцию [📖 Подробнее: Экспорт/Импорт](docs/export-import.md)

### Вопрос: Почему не работает полноэкранный режим?

**Ответ:** В настройках приложения можно выбрать режим: **Всегда**, **Никогда** или **Последнее состояние** (запоминать предыдущий режим).

### Вопрос: Можно ли сменить URL Lampa?

**Ответ:** Да, в настройках приложения есть поле **URL лампы**. По умолчанию: `http://lampa.mx`. Не рекомендуется менять без необходимости.

### Вопрос: Куда сохраняются настройки приложения?

**Ответ:**

- Windows: `%AppData%\Lampa\config.json`
- macOS: `~/Library/Application Support/Lampa/config.json`
- Linux: `~/.config/Lampa/config.json`

### Вопрос: Как полностью сбросить настройки?

**Ответ:** В меню приложения выберите **Настройки → Кеш и данные -> Очистить кеш -> Кеш и данные**.
